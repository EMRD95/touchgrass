# Assets and licenses

This project intentionally uses permissive/local assets rather than copying random proprietary indie-game sprites.

## Local reference image

- `public/assets/reference/grass_touching_simulator.webp`
- Source: user-provided reference image from the local project folder.
- Used as menu/background reference art only.

## Kenney CC0 assets

Downloaded from Kenney and copied into `public/assets/*`.
Each selected pack includes its original `License.txt` in the matching public asset folder.

- Tiny Town: https://kenney.nl/assets/tiny-town
  - License: Creative Commons CC0 1.0
  - Used for grass, flowers, path, tree tiles.
- Foliage Sprites: https://kenney.nl/assets/foliage-sprites
  - License: Creative Commons CC0 1.0
  - Used for touchable grass tufts, tinted in-game.
- Platformer Characters: https://kenney.nl/assets/platformer-characters
  - License: Creative Commons CC0 1.0
  - Used for player and doom-goblin character sprites.
- Impact Sounds: https://kenney.nl/assets/impact-sounds
  - License: Creative Commons CC0 1.0
  - Used for grass footsteps, touch, and hit sounds.
